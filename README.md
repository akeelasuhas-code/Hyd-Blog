# HyderaDay - Your One-Day Escape in Hyderabad

A visually stunning landing page for Suhas Akella, creator of HyderaDay. This website showcases a modern Japanese samurai aesthetic with sakura leaf animations, introducing the student-focused blog that helps students explore Hyderabad on a budget.

## Features

- **Japanese Samurai Theme**: Sharp outlines, modern asymmetrical layout, and calligraphic typography
- **Sakura Animations**: Floating cherry blossom petals with performance optimization
- **Responsive Design**: Optimized for mobile, tablet, and desktop viewing
- **Accessibility**: Semantic HTML5, ARIA labels, and keyboard navigation support
- **Performance Optimized**: Lazy loading, reduced motion support, and efficient animations

## Design Elements

- **Color Palette**: Dark gray (#2a2a2a), crimson (#8B0000), pure white (#ffffff), and soft pink (#FFB6C1)
- **Typography**: Cormorant Garamond, Playfair Display, Bebas Neue, and Roboto
- **Layout**: Asymmetrical grid design inspired by modern Japanese aesthetics
- **Animations**: Subtle floating sakura petals with respect to user preferences

## About HyderaDay

Created by Suhas Akella, HyderaDay is a student-focused blog that curates day-out plans, budget cafes, hidden gems, and free spots across Hyderabad. It's designed to help students explore more while spending less, making the most of what Hyderabad has to offer.
